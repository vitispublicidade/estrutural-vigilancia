# Estrutural Vigilância - Landing Page

Landing page profissional para Estrutural Vigilância e Segurança Ltda., desenvolvida com React, Tailwind CSS e design moderno.

## 🎯 Características

- ✅ Design responsivo e profissional
- ✅ Formulário de contato funcional
- ✅ Botão WhatsApp flutuante
- ✅ Animações sofisticadas
- ✅ Menu mobile responsivo
- ✅ Indicadores de seção ativa
- ✅ Galeria de imagens com carrossel
- ✅ Redes sociais integradas
- ✅ Otimizado para SEO

## 🛠️ Tecnologias

- **React 19** - Framework JavaScript
- **Tailwind CSS 4** - Estilização
- **Vite** - Build tool
- **Rajdhani + Inter** - Tipografia
- **Lucide React** - Ícones

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/vitispublicidade/estrutural-vigilancia.git
cd estrutural-vigilancia

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

O site estará disponível em `http://localhost:3000`

## 🚀 Build para Produção

```bash
npm run build
```

Os arquivos compilados estarão na pasta `dist/`

## 🌐 Deploy no Netlify

1. Acesse [netlify.com](https://netlify.com)
2. Clique em "Add new site" → "Import an existing project"
3. Conecte seu repositório GitHub
4. Configure:
   - **Build command**: `npm run build`
   - **Publish directory**: `dist`
5. Clique em "Deploy"

O site estará disponível em: `https://estrutural-vigilancia.netlify.app`
